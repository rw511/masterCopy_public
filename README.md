# masterCopy_public
Scripts to copy and rename commonly used collections of files.

Useful for creating quick test cases of whatever software you use. These few scripts and a folder can easily be tweaked to work for any commonly used files.

To use:
1) Move the masterCopyDeck to any folder you like, making sure to update the paths inside copy_deck.bat and rename_deck.bat
2) Move the masterCopy.bat file to a folder that is on your system path
3) Open a command prompt and type masterCopy
4) Enter the destination for your files
5) Enter the new name
6) Voilà
